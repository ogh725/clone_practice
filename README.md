# clone_practice
> clone 과 conflict 실습

1. 아래의 명령어를 통해 클론 생성
   ```bash
   git clone URL
   ```

2. local에서 commit을 하나 더 쌓은 다음 remote로 push

3. remote에서 commit을 하나 더 쌓은 다음 local에서 pull

4. commit을 쌓습니다.
 - local에서 commit을 하나 더 쌓는다.
 - remote에서 commit을 하나 더 쌓는다.
  
5. push를 하려고 했는데 rejected가 뜹니다.

6. 이를 해결하기 위해 pull

7. merge commit을 남깁니다.
